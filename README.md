🔎 Visualizador de Perfil do GitHub

Este projeto é uma aplicação web que permite buscar e visualizar informações públicas de usuários do GitHub através da API oficial do GitHub.

🚀 Funcionalidades

🔍 Buscar usuários do GitHub pelo nome de usuário

👤 Exibir foto de perfil

📝 Mostrar nome e biografia

👥 Exibir quantidade de seguidores

🕵️ Mostrar quantidade de usuários que a pessoa segue

📦 Listar os 10 últimos repositórios públicos

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript (ES Modules)

GitHub API

⚙️ Como usar

Digite o nome de usuário do GitHub no campo de busca.

Clique no botão de pesquisa ou pressione Enter.

As informações do perfil e os repositórios serão exibidos na tela.

📡 API utilizada

https://api.github.com/users/{username}
